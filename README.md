# Waywall wordle
Fun lil wordle game to play for Fun

## Structure:
- List of 5 letter words (get online or smth idk)

    - config.action to start "mode"
    - save original set of binds for going back
    - keep all qp, al, zm letters normal
    - config.actions on every letter
    - type 5 letters, mayyyybe press enter
    - compare 5 letters to chose word chosen randomly and saved
    - 6 guesses, stop if guessed
